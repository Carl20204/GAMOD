## MOD安装教程

* 下载安装「墨魔豆」

* 首次启动墨魔豆会出现【github访问代理设置】页面，没有魔法的话，选择镜像代理6

* 左滑进入【国际版MOD】页面，选择游戏

* 点击游戏图标，进入游戏详情页后，通过提供的链接下载官方原版安装包

* 确保安装包保存在手机「下载目录」

* 返回墨魔豆主页 → 点击 【添加游戏】

* 选择已下载的安装包进行安装

* 安装完成后，点击墨魔豆主页的游戏图标，左滑进入 【游戏设置】 点击 【下载来自GITHUB的MOD文件】（自动安装）

* 完成后即可启动游戏
